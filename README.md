# Number Guessing Game

A command-line number guessing game built in Python.

## Features
- Player sets own range using minimum and maximum values
- Player choosees how many tries they want
- Input validation and error handling for invalid inputs
- Higher/lower hints after each guess
- Tracks number of tries after each guess
- Asks player if they would like to play again

## How to Run
1. Make sure Python is installed on your computer
2. Download or clone this repository
3. Run the file in your terminal: python "Number_Guessing_Game.py"
